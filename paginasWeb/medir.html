Realizando mediciones

1. Conecte el sensor al dispositivo

2. Como se menciona en la guía 1, pruebe el funcionamiento del sensor

3. Una vez revise el sensor, puede realizar una prueba de medicó; entonces:

Ingrese a gráfica observe las medidas, luego con un encendedor o fósforo caliente el sensor y observe la variación en la gráfica

5. Cuando finalice, desconecte el sensor y apague el dispositivo
